Build a full enterprise-grade SaaS web application called "ProManageX".

The platform should be modern, scalable, responsive, secure, and built for large enterprises such as telecom companies, construction companies, outsourcing firms, engineering companies, logistics companies, and facility management providers.

Use:
- Frontend: React + TailwindCSS
- Backend: Node.js + Express
- Database: PostgreSQL
- Authentication: JWT + Role-Based Access Control
- File Storage: Local storage with future S3 compatibility
- Charts: Recharts
- UI Style: Modern enterprise dashboard similar to Monday.com, Jira, SAP Fiori, and ServiceNow
- Architecture: Modular scalable architecture
- API Style: RESTful APIs
- State Management: React Context or Zustand
- Tables: Advanced searchable/filterable data tables
- Notifications: Real-time notifications system
- Deployment Ready
